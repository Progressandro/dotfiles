## Setting up coding environment from a clean MacOS install:
2. Install Homebrew: https://brew.sh/
3. `brew install yadm`
4. Clone this repository using `yadm`.
5. Install Karabiner-Elements: https://karabiner-elements.pqrs.org/
6. Install yabai: https://github.com/koekeishiya/yabai/wiki/Installing-yabai-(latest-release) (Remember the tweaks)
7. Install skhd: https://github.com/koekeishiya/skhd?tab=readme-ov-file#install
8. Install Alacritty terminal: https://alacritty.org/
9. `xcode-select --install`
10. Install oh-my-zsh: https://ohmyz.sh/
11. `brew install zsh-autosuggestions`
13. `yadm restore .`
14. Install `FiraCode Nerd Font`: https://www.nerdfonts.com/font-downloads
15. `brew install tmux`
16. Install tpm: https://github.com/tmux-plugins/tpm (Remember to clone it to `~/.config/tmux/...` instead of `~/.tmux/...`
17. `Ctrl+B I` inside tmux
19. `brew install neovim`
20. Install `packer.nvim`: https://github.com/wbthomason/packer.nvim
21. `:PackerInstall` inside neovim
22. Set up a new github ssh key: https://docs.github.com/en/authentication/connecting-to-github-with-ssh/adding-a-new-ssh-key-to-your-github-account
